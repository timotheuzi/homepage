Privacy Policy for AutarkyLab

Effective Date: May 6, 2026

Owner: AutarkyLabs & timotheuzi@hotmail.com

Important Note on Data Storage

All data collected, processed, and stored by AutarkyLabs remains exclusively on your device. Nothing is stored on our servers or any external location other than your own device. We do not operate any servers for data storage or processing of user information. No data is saved outside of the phone.

1. Introduction

AutarkyLabs ("we", "our", "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our security monitoring application. All operations are performed locally on your device with no data transmission to external servers.

2. Information We Collect

2.1 Network Information

  • IP addresses of discovered devices
  • MAC addresses of network and Bluetooth devices
  • Device names and hostnames
  • Open port information
  • Network interface details
  • Vulnerability scan and security test results

2.2 Bluetooth Data

  • MAC addresses of nearby Bluetooth devices
  • Bluetooth signal strength (RSSI)
  • Device types and capabilities
  • Connection/disconnection events
  • BLE device information (including AirTags)

2.3 Hardware Sensor Data

  • Accelerometer readings
  • Gyroscope data
  • GPS coordinates and location information
  • Ambient light sensor data
  • Magnetometer readings
  • Battery status and charging information

2.4 System Information

  • CPU usage statistics
  • Memory usage data
  • Disk space information
  • Network interface statistics
  • Process information

2.5 Application Data

  • Threat detection logs
  • Security event history
  • User preferences and settings
  • Application usage logs for debugging

2.6 File System Scanner Data

  • File system structure and metadata
  • File content analysis results
  • Suspicious file patterns and signatures
  • File system change logs and timestamps

2.7 System Monitor Data

  • Performance metrics and historical trends
  • Resource usage patterns
  • System health indicators
  • Optimization recommendation logs

3. How We Use Your Information

We use the collected information exclusively for:

  • Security Monitoring: Detecting potential security threats and anomalies
  • Threat Detection: Identifying suspicious devices and activities
  • System Analysis: Monitoring system performance and health
  • User Interface: Displaying security status and alerts
  • Debugging: Improving application functionality and stability
  • File System Analysis: Identifying potential security risks in local files
  • Performance Optimization: Analyzing system performance for improvement recommendations

All usage occurs locally on your device.

4. Data Processing

4.1 Local Processing

All data processing occurs locally on your device. We do not transmit any data to external servers. If you choose to export logs, this is handled entirely by you and does not involve our servers.

4.2 Data Retention

  • Threat logs are stored locally until manually deleted
  • Sensor data is processed in real-time with minimal storage
  • File system scanner results are stored until the next scan
  • System monitor historical data is retained for trend analysis
  • No persistent data is retained beyond the application's runtime
  • Users can export or delete all collected data at any time
  • All retained data remains exclusively on your device

5. Data Sharing and Disclosure

5.1 No Third-Party Sharing

We do not share, sell, or rent your personal information to third parties. No data leaves your device.

5.2 System Permissions

The application requests location and Bluetooth permissions on initial install and requires system-level permissions to access:

  • Network interfaces for device discovery
  • Bluetooth adapters for signal monitoring
  • GPS for location-based security
  • Hardware sensors for environmental monitoring
  • File system for deep scanning
  • System resources for performance monitoring

5.3 Legal Requirements

We may disclose information if required by law or in response to valid requests by public authorities.

6. User Rights and Controls

6.1 Data Access

Users can view all collected data through the application interface.

6.2 Data Export

Users can export logs and collected data for external analysis. Exports are saved locally or to user-specified locations on your device or connected storage.

6.3 Data Deletion

Users can delete all collected data at any time through the application settings.

6.4 Feature Control

Users can disable any monitoring feature at any time:

  • Network scanning
  • Bluetooth monitoring
  • Sensor data collection
  • GPS tracking
  • File system scanning
  • System monitoring

6.5 Application Removal

Uninstalling the application completely removes all data from the device.

7. Security Measures

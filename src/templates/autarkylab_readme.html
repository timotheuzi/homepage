AutarkyLab 🛡️

A comprehensive, intelligent security monitoring application built with Flutter, designed for Android. This project provides real-time oversight of your digital and physical environment with AI-driven threat detection.

🚀 Overview

AutarkyLab provides real-time security monitoring of your network, Bluetooth environment, physical hardware sensors, file system, and system resources to protect you from digital and physical intrusions.

✨ Features

🔍 Network Security

  • Automatic Discovery: Scans local networks for new and disappeared devices.
  • Threat Detection: Identifies suspicious open ports and vulnerabilities.
  • Security Testing: Automated reconnaissance and vulnerability testing (XSS, SQL Injection, etc.).
  • WiFi Security: Detects Rogue Access Points, Evil Twin attacks, and unencrypted networks.

📡 Bluetooth Monitoring

  • Device Discovery: Monitors nearby Bluetooth devices and their signal strength.
  • AirTag Detection: Monitors BLE for Apple AirTags and Find My network-enabled devices.
  • Signal Analysis: Tracks RSSI history to identify potential relay attacks.
  • Trust Management: Mark devices as trusted to reduce false positive alerts.

🤖 AI Anomaly Detection

  • Pattern Learning: Learns from detections to improve confidence scores.
  • Connection Patterns: Detects unusually frequent connection attempts.
  • Threat Analysis: Provides confidence level and recommendations.

📁 File System Scanner

  • Deep Scanning: Scans for suspicious files and security risks.
  • Risk Assessment: Categorizes files by risk level (Low, Medium, High).
  • Pattern Detection: Identifies suspicious file patterns and signatures.
  • Selective Deletion: Review and delete suspicious files.

📊 System Monitor

  • Performance Tracking: Real-time CPU, memory, and disk usage monitoring.
  • Battery Health: Continuous status monitoring of power levels and charging states.
  • Network Connections: Monitor active network connections.
  • Historical Trends: Retained data for trend analysis.

📱 Hardware Sensor Monitor

  • Real-time Streams: Live monitoring of Accelerometer, Gyroscope, and Magnetometer.
  • Environment Sensing: Ambient light detection and precise GPS tracking.
  • Physical Security: Detects device movement and environmental changes.

📻 Radio Frequency Monitor (Coming Soon)

  • Will enable monitoring of radio frequencies for threats using SDR hardware.
  • Support for RTL-SDR and HackRF devices.

🎨 Beautiful Modern UI

  • Unified Dashboard: Real-time security status and quick statistics.
  • Dark/Light Mode: Full support for system-wide themes.
  • Detailed Logs: Categorized threat logs with severity-based color coding.

🛠️ Installation

Prerequisites

  • Flutter SDK (>= 3.0.0)
  • Android Studio / VS Code with Flutter extension

Build Commands

flutter pub get
flutter build apk --release

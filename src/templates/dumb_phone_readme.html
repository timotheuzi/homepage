Dumb Phone

A comprehensive network security application for Android and iOS that provides real-time protection against threats, monitoring network traffic, and blocking malicious connections through two distinct firewall modes.

Features

  • Dumb Phone Mode: High-security mode that blocks all non-essential traffic by default, allowing only voice and text-related communications.
  • Soft Firewall Mode: Intelligent monitoring and blocking based on customizable rules and threat intelligence.
  • Country Blacklisting: Block all traffic to/from specific countries known for high cyber-threat activity.
  • Real-time Logging: Monitor all allowed and blocked connections with detailed packet information.
  • Process Monitoring: View running processes and their associated network activity.
  • Custom Rules: Create your own IP, port, or protocol-based firewall rules.
  • Threat Intelligence: Pre-loaded database of high-risk countries and IP ranges.
  • Log Retention: Configurable log retention with automatic cleanup.

Privacy

All processing and logging happen locally on your device. Your network traffic data is never sent to external servers.
